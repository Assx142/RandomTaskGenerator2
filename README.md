# Random Task Generator

**Автор:** Лысенко Валерий

## Описание

Приложение для генерации случайных задач по категориям (учёба, спорт, работа). Позволяет добавлять собственные задачи, сохраняет историю генераций в JSON-файл, поддерживает фильтрацию истории и работает на Tkinter.

## Требования

- Python 3.6+
- Tkinter (входит в стандартную библиотеку)

## Установка и запуск

1. Сохраните код в файл `task_generator.py`
2. Запустите в терминале:
   ```bash
   python task_generator.py
